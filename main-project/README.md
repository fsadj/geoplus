# Main Project

这个目录用于承接与比赛/GEO 子项目无关的原项目主线代码、文档和依赖。

当前仓库历史演化中，可运行资产几乎都已经比赛化，因此本次重组优先完成了 `competition/` 的完整剥离与独立化。`main-project/` 先保留为清晰入口，后续如果你要继续恢复、迁入或重构原项目主线，可以从这里开始，而不再与比赛脚本和产物混放。

建议约定：

- 原项目自己的源码、脚本、文档和依赖都放在 `main-project/` 内部。
- 不要再直接依赖 `competition/outputs/`、`competition/data/` 或比赛命名约定。
- 如需长期保留通用能力，优先在未来以明确边界的方式从 `competition/` 中提炼，而不是重新把目录混回根级。
